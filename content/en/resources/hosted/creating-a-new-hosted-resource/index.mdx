## Introduction

This is an example of a Hosted Resource for the purposes of the DARIAH Campus Style Guide.  The initial section offers a brief introduction to the resource, outlining some of the key background principles that form its basis, and then offering some learning outcomes.

It can be broken down into a couple of paragraphs if necessary, but is it always formatted as 'normal text'.  There is also the option to use a 'Featured Image' at the top of this page as well, if it really adds to the pedagogical elements of the resource, however we advise caution here as featured images can also be a distraction or even an annoyance as users try to get to the information in the text.  We might therefore suggest using an image further down in the content if it is illustrative to the topic.

## Learning Outcomes

After completing this learning resource, users will be able to:

* Use the DARIAH-Campus Style Guide in their contributions to DARIAH-Campus
* Understand how the different formatting options within the DARIAH-Campus CMS can be combined
* Critique their own contributions to DARIAH-Campus against the Style Guide to ensure quality and conformity.

### More Widgets for You to Use

At this point in the resource we are not going to add any more text unless it is required to demonstrate a widget.

#### Link button

You can use this to add a link button out to an external website.

<LinkButton link={{"discriminant":"external","value":"https://www.dariah.eu"}}>
  Go to main DARIAH website
</LinkButton>

#### Quiz

There are a couple of quiz options ('multiple choice' or 'text input' options) available to you if you want to add short self-evaluation exercises into your learning resource.  You can do this mid-way through the resource to allow learners to check their learning as they go, or you can add quizzes at the end for a final test.

<Quiz>
  <QuizChoice variant="multiple">
    <QuizChoiceQuestion>
      What colour is the sky?
    </QuizChoiceQuestion>

    <QuizChoiceAnswer kind="incorrect">
      Red
    </QuizChoiceAnswer>

    <QuizChoiceAnswer kind="correct">
      Blue
    </QuizChoiceAnswer>

    <QuizChoiceAnswer kind="incorrect">
      Green
    </QuizChoiceAnswer>

    <QuizChoiceAnswer kind="incorrect">
      Yellow
    </QuizChoiceAnswer>

    <QuizSuccessMessage>
      Correct!  The sky is blue.
    </QuizSuccessMessage>

    <QuizErrorMessage>
      Sorry, that's not the right answer.  Try again!
    </QuizErrorMessage>
  </QuizChoice>
</Quiz>

#### Tabs

If you want to add a lot of related information but want to show it in a different format, tabs can offer a way to do that. &#x20;

<Tabs>
  <Tab title="South">
    Add some text here
  </Tab>

  <Tab title="East">
    More text here
  </Tab>

  <Tab title="West">
    Even more text here
  </Tab>

  <Tab>
    Fed up with seeing text?  Too bad, here's some more!
  </Tab>
</Tabs>

#### Video

When you add a video using the 'Video' widget, it directly embeds the video into the content.  This allows you to use videos in a similar way to how you might use figures or images.

#### Video Card

A Video Card acts a bit differently to a video.  In this case, it acts more as a link out to a video.  It allows you to add a caption for the video, as well as a preview image.

## Conclusion

All hosted resources should have a concluding paragraph or two.  The conclusion should offer a summary of what has been covered, and how this reflects the projected learning outcomes at the beginning of the resource.  This resource began by providing a demonstration of the layout and heading hierarchies within the Hosted Resource house style.  It then went on to showcase each of the widgets available to content creators when developing and drafting up their learning materials for publication.

This example resource can therefore act as a template of sorts that can help content providers to measure and critique their own resources in accordance with the house style.
